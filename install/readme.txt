Instructions to create and use the 'almos' conda environment from a YAML file

1. Save the YAML file
   - Copy the environment specification to a file named 'almos.yaml' in your working directory.

2. Open Anaconda Prompt
   - On Windows: Search for 'Anaconda Prompt' in the Start menu and open it.
   - On Linux/Mac: Open your preferred terminal.

3. Navigate to the folder where the YAML file is saved
   For example:
   cd C:\Users\YourUsername\YourFolder

4. Create the environment by running:
   conda env create -f almos.yaml

5. Wait for the installation to finish.

6. Activate the environment:
   conda activate almos
